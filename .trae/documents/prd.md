## 1. 产品概述

市场摊位租赁管理系统是一个面向市场管理方的全流程数字化管理平台，用于管理市场内摊位的租赁、租户信息、费用收缴及合同管理等核心业务。系统旨在替代传统手工台账，提升摊位出租率、降低管理成本、实现租赁业务的可视化和规范化。

- 目标用户：市场管理员、财务人员、市场负责人
- 核心价值：摊位信息数字化、租赁流程自动化、费用收缴透明化、数据统计实时化

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 超级管理员 | 系统初始化创建 | 全部功能，用户管理 |
| 市场管理员 | 管理员创建 | 摊位管理、租赁管理、租户管理、费用管理 |
| 财务人员 | 管理员创建 | 费用管理、报表查看 |

### 2.2 功能模块

1. **登录页**: 用户认证、Cookie会话保持
2. **仪表盘**: 数据概览、关键指标、待办事项
3. **摊位管理**: 摊位列表、摊位详情、区域划分、状态管理
4. **租户管理**: 租户信息、联系方式、租赁历史
5. **租赁管理**: 合同签订、续租、退租、合同到期提醒
6. **费用管理**: 租金收缴、水电费、欠费提醒、缴费记录
7. **系统设置**: 用户管理、角色权限、系统参数

## 3. 核心流程

### 3.1 摊位租赁流程

用户登录系统后，在摊位管理中查看空置摊位，选择摊位后为指定租户签订租赁合同，系统自动生成首月租金账单。租户按月缴费，财务人员在费用管理中登记缴费。合同到期前系统自动提醒，管理员可操作续租或退租。

```mermaid
flowchart TD
    "管理员登录" --> "查看空置摊位"
    "查看空置摊位" --> "选择摊位签订合同"
    "选择摊位签订合同" --> "填写合同信息"
    "填写合同信息" --> "合同生效"
    "合同生效" --> "自动生成租金账单"
    "自动生成租金账单" --> "租户缴费"
    "租户缴费" --> "财务登记缴费"
    "合同到期前" --> "到期提醒"
    "到期提醒" --> "续租"
    "到期提醒" --> "退租"
    "退租" --> "摊位释放为空置"
```

### 3.2 费用收缴流程

```mermaid
flowchart TD
    "月初批量生成账单" --> "租户收到缴费通知"
    "租户收到缴费通知" --> "按时缴费"

    "财务登记" --> "账单状态更新为已缴"
    "租户收到缴费通知" --> "未按时缴费"
    "未按时缴费" --> "逾期提醒"
    "逾期提醒" --> "补缴缴费"
    "补缴缴费" --> "财务登记"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深青色 (#0F766E) + 暖橙色 (#EA580C) 作为强调色
- 辅助色：石板灰 (#475569) 用于文字，浅灰 (#F1F5F9) 用于背景
- 按钮风格：圆角 (rounded-lg)，主按钮实色填充，次按钮描边
- 字体：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- 布局风格：左侧固定导航栏 + 顶部标题栏 + 右侧内容区
- 图标风格：线性图标 (Lucide)

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录页 | 登录表单 | 居中卡片式布局，左侧装饰图案，右侧表单，深色背景 |
| 仪表盘 | 数据卡片 | 4列网格卡片，图标+数字+趋势箭头，卡片微阴影 |
| 仪表盘 | 待办提醒 | 列表卡片，左侧色条标识紧急程度，右侧操作按钮 |
| 仪表盘 | 趋势图 | 折线图，渐变填充区域，工具提示 |
| 摊位管理 | 摊位列表 | 表格布局，状态标签彩色圆点，行悬浮高亮 |
| 摊位管理 | 摊位详情 | 顶部信息卡+下方Tab切换（租赁历史/费用记录） |
| 租户管理 | 租户列表 | 卡片+表格混合，头像占位，快捷操作按钮 |
| 租赁管理 | 合同列表 | 表格布局，状态标签不同颜色，到期倒计时 |
| 费用管理 | 费用列表 | 表格布局，金额右对齐，逾期标红，批量操作栏 |
| 系统设置 | 用户管理 | 表格布局，角色标签，启用/禁用开关 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1024px
- 导航栏在移动端收起为汉堡菜单
- 表格在窄屏下支持横向滚动
- 卡片网格自适应列数

### 4.4 动效设计

- 页面切换：淡入淡出
- 列表加载：骨架屏占位
- 按钮交互：悬浮微放大 + 阴影加深
- 数据变化：数字滚动动画
- 通知提示：右上角滑入滑出
